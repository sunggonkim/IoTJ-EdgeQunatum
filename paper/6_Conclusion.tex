\section{Conclusion}
\label{sec:conclusion}

This paper presented \EdgeQuantum, a framework enabling variational quantum algorithms on resource-constrained IoT edge devices. Through systematic evaluation on NVIDIA Jetson Orin Nano, we established the following key findings:

\begin{enumerate}
    \item \textbf{Scalability Achievement:} Using tiered memory offloading (GPU$\rightarrow$DRAM$\rightarrow$Storage), edge devices achieve \textbf{34-qubit simulation} (17.1 billion amplitudes). Native cuQuantum acceleration coupled with LZ4 compression enables this scale on 15W hardware with a 33\% performance improvement over non-native implementations.
    
    \item \textbf{VQE Feasibility:} The transverse-field Ising model is solvable up to 20 qubits with 9.15 seconds per optimization iteration, demonstrating practical ground state estimation.
    
    \item \textbf{QAOA Accuracy:} MaxCut QAOA achieves 100\% approximation ratio for graphs up to 10 nodes with a single variational layer.
    
    \item \textbf{Energy Efficiency:} Operating at 15W, edge quantum simulation achieves 26$\times$ better energy efficiency compared to datacenter GPU alternatives.
\end{enumerate}

These results establish the viability of quantum-inspired edge computing for distributed IoT applications. Our tiered memory approach, inspired by AURORA-Q, demonstrates that storage offloading with compression can extend quantum simulation far beyond physical memory limits. Future work will explore asynchronous I/O pipelines and tensor network methods for further scaling.
